 INTRODUCTION
 ------------
 Drupal Origins Extra modules extension (drupal_origins_extra) is a basic
 module which enables some extra drupal modules, and their dependencies. That
 modules has no a concrete focus, but they are very frequent at drupal sites.

 This module doesn't set any additional default settings during the install
 process. For more info about which modules are enabled with
 drupal_origins_extra, please see the section "Enabled modules" on this file.

 ENABLED MODULES
 ---------------
 As it was said at "Introduction" section, Drupal Origins Extra Extension enables
 several modules. That modules are very frequent at Drupal sites, such as
 jCarousel or Breakpoints modules. The full list of modules enabled with Drupal
 Origins Extra are the following:

  * Flexslider (https://www.drupal.org/project/flexslider)
  * Flexslider Views (Flexslider submodule for Views integration)
  * jCarousel (https://www.drupal.org/project/jcarousel)
  * Colorbox (https://www.drupal.org/project/colorbox)
  * Chosen (https://www.drupal.org/project/chosen)
